# Birthday — стилистическое преобразование чисел

Консольная программа на Python, которая запрашивает дату рождения пользователя и выводит информацию о ней: день недели, високосность года, возраст и дату в виде «электронного табло» из звёздочек.

## Возможности

- Ввод даты рождения: день, месяц и год
- Определение дня недели для введённой даты
- Проверка, был ли год високосным
- Расчёт текущего возраста в полных годах
- Вывод даты в формате `дд мм гггг`, где цифры прорисованы символами `*`

## Требования

- Python 3.8 или новее
- Стандартная библиотека Python (внешние зависимости не нужны)

## Запуск

```bash
python Main.py
```

Программа последовательно запросит:

1. День (1–31)
2. Месяц (1–12)
3. Год (1900 — текущий год)

При некорректной дате (например, 31 апреля) программа сообщит об ошибке и попросит ввести дату заново.

## Пример работы

**Ввод:**

```
День (1-31): 29
Месяц (1-12): 2
Год (1900-2026): 2000
```

**Вывод:**

2000 год — високосный.
Сейчас вам полных лет: 26

Ваша дата рождения на электронном табло:

*****  *****         *   *  *****         *****  *   *  *   *  *   *
*          *         *   *  *             *      *   *  *   *  *   *
*          *         *   *  *             *      *   *  *   *  *   *
